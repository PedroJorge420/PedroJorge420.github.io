Alterar Dados da Conta - Versão 1

Atores

Cliente

Pre-condição

O usuário deve ter feito "log-in" e obtido autorização do sistema

Fluxo de Alterar Conta Primario (Caminho Basico)

  1. O caso de uso se inicia quando o usuário clica na opção "Minha conta"
  2. O usuario seleciona "Alterar Conta".
  3. O usuario faz a modificações nos campos desejados.
  4. O usuario envia as informações modificadas. (Veja Alternativa)
  5. As informações são validadas.

Pós-condição

A conta do usuario é alterada

Alterar Dados da Conta - Versão 2

Atores

Cliente

Pre-condição

O usuário deve ter feito "log-in" e obtido autorização do sistema

Fluxo de Alterar Conta Secundario (Caminho Alternativo)

  1. O usuario seleciona "Alterar Conta".
  2. O usuario faz a modificações nos campos desejados.
  3. O usuario envia as informações modificadas.
    • Se no passo 3 o usuario enviar as informações com dados não correspondentes aos campos certos, ja existentes, ou campos não prenchidos, o sistema informa ao usuario a preencher corretamente os dados que faltam.

Pós-condição

A usuario continua alterando sua conta

Fazer pedido - versão 1

Atores

Cliente

Pré-condição

O cliente ser cadastrado no site

Fluxo de eventos(Caminho básico):

  1. O caso de uso se inicia quando o cliente seleciona o produto
    • Se o usuário não for cadastrado, redireciona-lo para a pagina de login
  2. O sistema informa as informações do produto
  3. O cliente informa se deseja adicionar o produto ao carrinho ou continuar a compra(alternativa)
  4. O cliente informa a quantidade do produto que deseja comprar
  5. O cliente informa os dados do cartão para a compra
  6. O sistema verifica as informações, cria um pedido e envia as informações de pagamento ao sistema de contabilidade
  7. Quando o pagamento é confirmado, um id para visualizar o pedido é fornecido ao usuário e o pedido é despachado

Pós-condição

O pedido é enviado

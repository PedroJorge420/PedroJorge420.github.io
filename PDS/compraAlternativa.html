Fazer pedido - versão 2

Atores

Cliente

Pré-condição

O cliente ser cadastrado no site

Fluxo de eventos(Caminho secundário):

  1. O caso de uso se inicia quando o cliente seleciona o produto
    • Se o usuário não for cadastrado, redireciona-lo para a pagina de login
  2. O sistema informa as informações do produto
  3. O cliente informa se deseja adicionar o produto ao carrinho ou continuar a compra
    • O produto é adicionado ao carrinho e o usuário volta para a página de produtos

Pós-condição

O usuário continua a compra

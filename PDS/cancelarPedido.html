Fazer pedido - versão 1

Atores

Cliente

Pré-condição

O cliente ser cadastrado no site
O cliente deve ter feito pedido

Fluxo de eventos(caminho básico):

  1. O caso de uso começa quando o cliente clica em "Meus pedidos"
  2. O cliente seleciona o pedido que deseja excluir
  3. O sistema verifica se o pedido de cancelamento foi feito em um prazo de 3 dias desde o pedido
  4. Se estiver dentro do prazo, o pedido é cancelado

    Fluxo de eventos(caminho secundário)

    Se o pedido de cancelamento for feito após 3 dias do pedido, o pedido não poderá mais ser cancelado.

Pós-condição(caminho básico)

O pedido é cancelado

Pós-condição(caminho secundário)

O pedido não é alterado

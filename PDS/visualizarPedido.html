Fazer pedido - versão 1

Atores

Cliente

Pré-condição

O cliente ser cadastrado no site
O cliente deve ter feito pedido

Fluxo de eventos:

  1. O caso de uso começa quando o cliente clica em "Meus pedidos"
  2. O cliente pesquisa o pedido em questão ou pesquisa com o id
    • Se o id informado não existir, uma mensagem é exibida dizendo que o pedido não existe
  3. O sistema mostra as informações do pedido em questão

Pós-condição

O pedido é exibido, mas não é alterado

Fluxo de Cadastro Secundario (Caminho alternativo)

  1. O usuario acessa a aba de cadastro
  2. O usuario começa a prencher os campos sobre suas informações corrretamente
  3. O usuario, após o termino, envia as informações ao banco de dados
    • Se no passo 3 o usuario enviar as informações com dados não correspondentes aos campos certos, ja existentes, ou campos não prenchidos, o sistema informa ao usuario a preencher corretamente os dados que faltam.

Pós-condição

A usuario continua a fazer seu cadastro
